# Screenshots

This folder is for the required screenshots of the running application:

- `login.png` — Login Page
- `register.png` — Registration Page
- `dashboard.png` — Dashboard (Admin/Manager/Instructor/Student)
- `profile.png` — User Profile
- `role-management.png` — Role Management
- `mobile-view.png` — Mobile view (showing the hamburger menu open)

Since this submission was assembled without a live database connection,
add your own screenshots here after running the app locally (see the
root `README.md` for setup steps). Most image viewers/markdown renderers
will pick up files placed in this folder automatically once named as above.
